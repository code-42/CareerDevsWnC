I upgraded the basic version of the program by adding console.log() after each line of the program, to display the results of each line that adds, changes or deletes an item in the array.

so now my program looks like this:

// store todos
var todos = ['item1','item2','item3'];

// display todos
console.log('My todos:', todos);

// add new todos
todos.push('item 4');
console.log('My todos:', todos);
todos.push('item 5');
console.log('My todos:', todos);
todos.push('new todo');
console.log('My todos:', todos);

// changing the first item
todos[0] = 'item 1 updated';
console.log('My todos:', todos);
todos[0] = 'changed!';
console.log('My todos:', todos);

// Deleteing the first item
todos.splice(0,1);
console.log('My todos:', todos);
